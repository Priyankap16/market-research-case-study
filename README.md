# market-research-case-study

Competitor Analysis of Food Delivery Apps in Qatar

Description:
This repository contains an independent market research case study analyzing the competitive landscape of food delivery applications in Qatar.

Objective:
To compare key competitors and identify differences in features, pricing, positioning, and user perception.

Methodology:
- Secondary (desk) research
- Review of competitor websites and apps
- Analysis of publicly available user reviews and pricing information

Competitors Analyzed:
- Talabat
- Snoonu
- Deliveroo

Deliverable:
- PDF report with findings, insights, and recommendations
